# Provve Backend

## Модули системы
- **notification**: Управление уведомлениями и e-mail рассылками.
- **validation**: Принятие решений, расчёт итоговых результатов, интеграция с антифрод системой и контейнеризация.
- **payment**: Интеграция платежного шлюза, обработка заявок на оплату, передача уведомлений в модуль оплаты.
- **skill**: Опериррование навыками, экаменами и голосованиями (отдаёт команды другими модулями по результату голсования).
- **api**: Обработка запросов верхнего уровня, разделение на подзадачи для отдельных модулей, без встроенной бизнес-логики.
- **account**: Оперирование аккаунтами, разграничение доступа.

## Архитектура

1. Вся система построена по структуре модульного монолита.
2. Каждый из модулей содержит всю логику своей части: от контроллеров до контекста домена. Это позволит развернуть
   модули как самостоятельные микросервисы в будущем, когда потребуется масштабировать одну из функциональностей.
